# VOIDWHISPER-SPACE-SHOOTER

## Game Description

VoidWhishper- Space Shooter is a fast 2D space shooter game. You control a spaceship and fight enemy ships in space. The game is quick and action-packed, with waves of enemies and big boss fights. It has cool space backgrounds and exciting sound effects.

## Features

❖	Shooting: 
▪	You shoot automatically. Upgrades make your shots stronger or faster.

❖	Enemies: 
▪	Many types of enemies with different attack patterns.

❖	Boss Fights: 
▪	Big enemies at the end of each level with special moves.

❖	Health System: 
▪	You have health and can collect shields for protection.

❖	Levels: 
▪	The game gets harder step by step.

❖	Scoring: 
▪	You earn points for killing enemies, beating bosses, and picking up power-ups.          Bonus points for doing well without getting hit.
